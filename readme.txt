== Description ==

This widget will add a new workout to your blog each day. All [workout routines](http://www.workoutbox.com/workouts/) are created by the personal training team at WorkoutBOX.

WorkoutBOX Features professionally designed [muscle building workouts](http://www.workoutbox.com/workouts/muscle-building-workouts/) and [fat burning workouts](http://www.workoutbox.com/workouts/fat-burning-workouts/) with step-by-step video instructions, combined with personalized support from in-house fitness team.

[Download Now!](http://downloads.wordpress.org/plugin/workout-of-the-day.zip)

== Installation ==

1. You can either use the automatic plugin installer or your FTP program to upload it to your wp-content/plugins directory the top-level folder. Don't just upload all the php files and put them in /wp-content/plugins/.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the Workout of the Day widget to your sidebar through the 'Appearance => Widgets' menu in WordPress. 
4. That's it!
